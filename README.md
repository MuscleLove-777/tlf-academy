# TLF Academy - 臨床試験TLFプロ育成プラットフォーム

臨床試験の **Tables, Listings, Figures（TLF）** を基礎からプロレベルまで体系的に学べる日本語学習プラットフォームです。

## 概要

TLFは臨床試験における統計解析のアウトプットであり、Clinical Study Report（CSR）の中核を成す成果物です。本アカデミーでは、TLFの基礎概念から、SAS/Rによるプログラミング実践、品質管理、規制要件まで全13レベル・52モジュールで網羅的に学習できます。

## カリキュラム（全13レベル）

| Level | タイトル | 内容 |
|-------|---------|------|
| 1 | TLFの基礎 | 定義・目的・CSRとの関係・キャリアパス |
| 2 | 臨床試験の基礎知識 | フェーズ・試験デザイン・SAP・規制当局 |
| 3 | CDISC標準の概要 | CDASH→SDTM→ADaM→TLFの流れ |
| 4 | ADaMデータセットの理解 | ADSL, ADAE, ADLB, ADTTE |
| 5 | TLFの設計と仕様書 | Shell/Mock TLF・プログラミング仕様書 |
| 6 | Tables（表）の基礎 | 人口統計表・ベースライン・安全性要約 |
| 7 | Tables（表）の応用 | 有効性解析・検査値・サブグループ |
| 8 | Listings（リスト） | AE・検査値・併用薬リスト |
| 9 | Figures（図） | KM曲線・Forest Plot・箱ひげ図 |
| 10 | SASによるTLFプログラミング | PROC REPORT・ODS・マクロ |
| 11 | RによるTLF入門 | pharmaverse・rtables・ggplot2 |
| 12 | 品質管理と検証 | 二重プログラミング・PROC COMPARE |
| 13 | 規制要件と提出プロセス | ICH E3・eCTD・最新トレンド |

## 対象者

- 製薬会社・CROのSASプログラマー
- 統計解析担当者
